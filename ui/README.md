# CodeReview UI

Фронтенд на базе React + Vite. Предоставляет экран списка запусков и карточку запуска с фильтрами по findings и заглушкой LLM этапа.

## Скрипты

```bash
npm install
npm run dev   # локальная разработка
npm run build # сборка для prod
```

Переменные окружения:
- `VITE_API_BASE` — URL бэкенда (по умолчанию `http://localhost:8000/api`).
