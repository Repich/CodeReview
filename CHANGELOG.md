# Changelog

## 2026-01-27
- Добавлена настройка пользователя «Использовать все нормы».
- В пайплайн LLM добавлены отдельные шаги для критических норм и норм из norms.yaml.
- Результаты паттернов/критических/обычных норм теперь объединяются в LLM с удалением дублей.
- Артефакты LLM получили новые типы стадий: critical, norms, merge.
- В UI добавлен просмотр артефактов (.txt/.json) в модальном окне с подсветкой JSON.
- Добавлена статическая норма: &НаСервере без обращения к Объект/ЭтаФорма/ЭтотОбъект.
- Добавлена статическая норма: запрет вывода сообщений пользователю в &НаСервере (кроме событий объекта).
- На форме создания запуска тип модуля теперь выбирается из списка.
- Шаг selection теперь делится на два запроса; лимит промпта увеличен до 100k символов.
- В UI добавлен просмотр исходников из sources_raw.zip с выбором файла и просмотром содержимого.
- Для админов добавлен экран «Проверка детерминизма» с JSON-отчётом по повторяемости выбора норм.
- Исправлено падение access-log middleware при detached user (без сессии).
- Исправлена запись audit_log для evaluation-запуска (flush перед вставкой).
- В payload результата воркера добавлен evaluation_report.
- Парсер selection теперь понимает формат с applicable/evidence (объекты), а не только список norm_id.
- В selection добавлены подсказки детекта (маркеры) из норм.
- В просмотре артефактов и исходников добавлена кнопка «Копировать в буфер».
- В режиме детерминизма добавлено сравнение: baseline vs prefilter (детерминированный предфильтр норм).
- В UI проверки детерминизма добавлен итог «стало лучше/хуже» по сравнению baseline vs prefilter.
- В worker добавлена поддержка провайдера OpenAI (gpt-5-mini) через настройки LLM_PROVIDER и OPENAI_APIKEY.
- В личном кабинете (только для админа) добавлен выбор LLM провайдера и модели.
- Ограничено изменение LLM провайдера/модели только для админов.
- Исправлено падение worker при evaluation: прокинуты llm_provider/llm_model в selection.
- Исправлена загрузка ключа LLM: теперь берется под провайдера из настроек запуска.
- Для OpenAI отключена передача temperature (модель gpt-5-mini принимает только дефолт).
- Кнопка «Перезапустить» на evaluation-запуске теперь перезапускает базовый запуск, а не evaluation.
- В обычных запусках включен предфильтр норм перед selection.
- Минимальный размер code unit увеличен до 100 строк (малые процедуры объединяются).

## 2026-01-22
- Добавлены настройки интерфейса (раздельно/вместе).
- Добавлено окно с changelog в личном кабинете.
- В объединённом режиме статические нарушения показываются в левом списке с LLM.
- Клик по статической карточке подсвечивает соответствующий код.
- Добавлена маскировка строковых литералов перед отправкой в LLM с отчётом в артефактах.
- Добавлено маскирование комментариев перед отправкой в LLM (отчёт в артефактах).
- Добавлено правило: не более 5 параметров у процедуры/функции (minor).
- Добавлена настройка пользователя для отключения проверки паттернов.
- Добавлены ручные нормы CUSTOM_003..CUSTOM_006 (безопасные расширения, уникальные ключи, проверки структуры).
- LLM теперь сначала выбирает релевантные нормы по названиям, затем проверяет только их.
- Лимит выбора норм увеличен до 40k символов; артефакты шага выбора помечаются как select.
